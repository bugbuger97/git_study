깃허브 - 다른 컴퓨터로, 기존 계정의 원격 저장소에 커밋하기

' clone ' 깃 명령어를 통해,
원격 저장소의 데이터를 현재 컴퓨터로 복제할 수 있다.
먼저, 원격 저장소에서 사용하던 이름과 계정을 현재 컴퓨터에 등록시킨다.

1. 기존 원격저장소 이름과 계정을 모르겠을 때
기존 컴퓨터에서 Terminal을 열어 아래 명령어를 입력한다. 

git config --list

입력하면 기존 컴퓨터에 등록된 계정 정보가 나타난다.

2. user.name과 user.email 정보를 현재 컴퓨터에 등록한다.

git config --global user.name [name]
git config --global user.email [email]

이 과정이 끝나고 저장할 디렉토리로 이동하여 아래 명령어를 입력하면
해당 디렉토리에 원격 저장소에 있는 데이터가 복제된다. 

( git clone + 원격저장소url )
git clone https://github.com/유저이름/원격저장소이름 

완료되었다는 메시지가 뜨면, 디렉토리에 복제된 폴더를 확인할 수 있다. 
생성된 폴더로 디렉토리를 변경하고 git log를 확인해 보면,
git을 포함한 폴더 전체가 잘 복제된 것을 확인할 수 있다. 

이 과정이 완료되면, 현재 컴퓨터에서 기존 컴퓨터와 동일하게 작업할 수 있게 된다.
현재 컴퓨터에서 변경된 내용을 commit 한 이후에 push 하면 원격 저장소에 이어서 저장된다. 
그리고 기존 컴퓨터에서 다시 작업을 하려면, 변경된 내용을 아래와 같이 pull 해줘야 한다. 

git pull origin main
